# 智慧燈具前導判斷表單

## 1. 既有開關處理
- [ ] 保留原有開關
  - [ ] 加裝智慧通斷器
  - [ ] 加裝智慧調光器
  - [ ] 更換智慧燈泡
- [ ] 不保留，直接更換智慧開關
- [ ] 需增補網路線（毋須增補中性線）

---

## 2. 天花板類型與線路

### (A) 水泥天花（預埋八角盒）
- [ ] 火線(L)：1 條（紅/黑）
- [ ] 中性線(N)：1 條（白）
- [ ] 控制線/燈線(L1/L2/L3)：1~3 條
- [ ] 雙切線：2 條同色連絡線

### (B) 木作/矽酸鈣天花
- [ ] 中性線(N)：1 條（白）
- [ ] 控制線/燈線(L1/L2/L3)：1~3 條
- [ ] 無火線(L)
- [ ] 有維修孔（利於補線）

### (C) 輕鋼架天花
- [ ] 配線彈性大，依現場評估

---

## 3. 牆面類型與線路

### (A) 水泥牆/磚牆/輕隔間（預埋單/雙聯盒）
- [ ] 火線(L)：1 條（紅/黑）
- [ ] 控制線/燈線(L1~L6)：1~6 條
- [ ] 雙切線：2 條同色連絡線（火線或控制線擇一）
- [ ] 盒內無中性線(N)（老屋可能有）
- [ ] 單聯盒：1~3 切
- [ ] 雙聯盒：2~6 切

### (B) 木作牆/矽酸鈣牆
- [ ] 中性線(N)：1 條（白）
- [ ] 控制線/燈線(L1~L3)：1~3 條
- [ ] 線路空間大，布線彈性高
